# BiloaLoginProject

A Java Web MVC Login Application with Servlets and JSP

## Features
- User login with database authentication
- User registration
- MVC architecture
- MySQL database integration

## Technologies Used
- Java Servlets
- JSP
- MySQL
- Apache Tomcat 11
- Eclipse IDE

## Setup Instructions
1. Import project into Eclipse
2. Configure Tomcat 11 on port 8085
3. Create MySQL database "market" with table "login"
4. Add MySQL connector JAR to build path
5. Run on server

## Database Configuration
- Database: market
- Table: login(username VARCHAR(50) PRIMARY KEY, userpwd VARCHAR(50))
- Test users: (PSG/FRANCE), (REAL/SPAIN), (CANON/CAMEROON)
